qsort - Рекурсивная функция производящая быструю сортировку, количество входных парметров функции - 3

Parameters
mass– параметр, вводящий массив, который необходимо отсортировать
low- параметр, вводящий нижнюю границу массива для сортировки
high- параметр, вводящий верхнюю границу массива для сортировки

Выбираем опорный элемент - середину массива

Выполняем цикл с постусловием: следим за текущими индексами, пока индексы стоят на своих местах

Сравниваем элементы нижней границы с опорным элементом, если элементы стоят верно, то инкрементируем индекс

Сравниваем элементы верхней границы с опорным элементом, если элемент стоят верно, то индекс декрементируем

Если индекс верхней границы стал больше, чем индекс нижней границы, значит произошло что-то не то и элементы нужно поменять

Если итоговый индекс верхней границы больше начального параметра нижней границы, то вызываем функцию qsort(mass,low,j)

Если итоговый индекс нижней границы меньше начального параметра верхней границы, то вызываем функцию qsort(mass,low,j)
